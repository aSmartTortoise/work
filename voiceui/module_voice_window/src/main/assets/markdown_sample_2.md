1.  使用遥控钥匙：长按智能遥控钥匙的锁止/解锁按键，车窗玻璃将自动上升。
2.  中控屏设置：进入中控屏的“门窗”选项，选择“车窗”界面，设置开启锁车自动升窗功能。
3.  使用岚图汽车APP：您也可以通过岚图汽车A窗的开启/关闭。
4.  车载智能语音控制：通过车载智能语音控制车窗的开启/关闭。
